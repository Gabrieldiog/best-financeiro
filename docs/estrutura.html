💰 Best Financeiro

Sistema de Gestao Financeira via Terminal

Java 21 Terminal SQLite

📋 Visao Geral do Sistema

O Best Financeiro e um sistema de gestao financeira pessoal desenvolvido em Java, que opera inteiramente via linha de comando (terminal). Cada usuario tem seu proprio login e gerencia suas receitas e despesas de forma isolada, com saldo, relatorios, projecao de gastos, sugestao de economia e exportacao de extrato em CSV.

ℹ️ Tipo de Aplicacao: Aplicacao de Console (CLI) com persistencia em banco relacional SQLite via JDBC.

Menu Inicial

Ao iniciar, o sistema pede autenticacao. A senha e armazenada com hash, nunca em texto plano.

[1] Cadastrar
Cria um novo usuario com validacao de senha forte.
[2] Login
Autentica o usuario e abre o menu principal.
[3] Sair
Encerra o programa.

Menu do Usuario (apos login)

[1] Registrar movimentacao
Cadastra receita ou despesa com tipo, valor, quantas vezes por mes acontece, categoria, descricao e data. Da pra criar categorias novas na hora.
[2] Listar movimentacoes
Mostra todas as movimentacoes do usuario.
[3] Editar movimentacao
Altera uma movimentacao existente pelo id.
[4] Excluir movimentacao
Remove uma movimentacao (com confirmacao).
[5] Ver saldo
Exibe total de receitas, despesas e saldo atual.
[6] Relatorios
Totais e gastos por categoria com percentual.
[7] Projecao de gastos do mes
Estima o gasto total do mes com base no ritmo atual.
[8] Sugestao de economia
Aponta a categoria de maior gasto e quanto da pra economizar.
[9] Exportar CSV
Gera um arquivo de extrato com todas as movimentacoes.
[10] Logout
Volta para o menu inicial.

🚀 Como Executar

Pre-requisitos

  • Java JDK 17 ou superior (testado no JDK 21)
  • Terminal / Prompt de comando

O driver do SQLite ja vem na pasta lib/ e o banco e criado sozinho no primeiro uso.

Comandos

# Linux / Mac
./run.sh

# Windows
RunInWindows.bat
ℹ️ Build manual: javac -cp "lib/*" -d out $(find src/main/java -name "*.java") && java -cp "out:lib/*" App
